# CSRF (Cross-Site Request Forgery) 

## 🚨 What is CSRF?

CSRF is a **security attack** where a malicious website tricks a **logged-in user** into performing unwanted actions on another website.

### 💡 Example
1. User logs into bank (session active)
2. Opens malicious website
3. That site sends request:
   POST /transfer-money
4. Browser automatically sends cookies → request succeeds

👉 User didn’t intend it, but action happens.

---

## 🛡️ How Spring Boot Prevents CSRF

Spring Boot (via Spring Security) uses a **CSRF Token**.

### 🔐 CSRF Token
- Random secret token generated by server
- Sent to frontend
- Must be included in every request (POST, PUT, DELETE)
- Server verifies token before processing request

If token is:
- Missing ❌
- Invalid ❌  
  → Request is rejected

---

## 🔄 Flow

1. User loads page
2. Server generates CSRF token
3. Token sent to frontend (form/response/header)
4. Frontend sends token with request
5. Server validates token
6. If valid → request allowed ✅

---

## ❓ Why GET Method Does NOT Require CSRF Token

GET requests are **safe (read-only)** operations.

### Reasons:
- GET should NOT change server data
- Used only for fetching data
- Even if attacker triggers GET → no damage expected

### Example:
GET /get-user-data → safe  
POST /delete-user → dangerous ⚠️

👉 CSRF protection is applied only to:
- POST
- PUT
- PATCH
- DELETE

---

## ⚙️ Default Behavior in Spring Boot

- CSRF is **enabled by default**
- Works for **session-based authentication**

---

## 🔌 How to Get CSRF Token from Server

### 1️⃣ Using Thymeleaf (Server-side rendering)

```html
<input type="hidden" name="_csrf" value="${_csrf.token}" />
```

## 2️⃣ Getting CSRF Token from Server (for React / Frontend)

Spring Security provides CSRF token as request attributes:

- `_csrf.token`
- `_csrf.headerName`

You can expose it using a controller:

```java
@GetMapping("/csrf-token")
public CsrfToken getCsrfToken(HttpServletRequest request) {
    return (CsrfToken) request.getAttribute("_csrf");
}
```

## ✅ Example Response

```json
{
  "headerName": "X-CSRF-TOKEN",
  "parameterName": "_csrf",
  "token": "random-token-value"
}
```

## ️⃣ Sending CSRF Token from React / Frontend
```javascript
fetch("/api", {
method: "POST",
headers: {
"X-CSRF-TOKEN": token
},
body: JSON.stringify(data)
});
```

## ❌ Disabling CSRF (for JWT / Stateless APIs)
```
http.csrf().disable();
```

# When to Disable CSRF
Using JWT authentication
No session (stateless backend)
Pure REST APIs
